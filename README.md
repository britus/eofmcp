## EoF MCP Server

### Overview

The project contains a MCP Server based on [QTMCP](https://github.com/britus/eofmcp/blob/master/README_qtmcp.md) 

### Dependencies

- QT-6 framework and its dependencies.

# Source Code Handler Tool

## Overview
The Source Code Handler tool is designed to facilitate managing and processing source code in a structured manner.

## Features
- **Local File Management**: Allows users to list, read and write source files efficiently.

## Requirements
- Latest QT-6 framework

## Installation
1. Clone the repository: `git clone https://github.com/britus/eofmcp.git`
2. Follow the setup instructions in the repository to configure the tool for your environment.
