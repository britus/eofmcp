# Complete ready to go solution

## EoF MCP Studio for macOS

This is a Model Context Protocol (MCP) server app for macOS

Powerful plugin management, scripting environment, and MCP server for 
modern AI based development workflows. Everything you need to manage 
your MCP tools efficiently

## Features Implemented

- Full MCP protocol implementation (version 2025-06-18)
- HTTP transport layer
- Three core services: Tools, Resources, Prompts
- Configuration-driven startup
- Flexible tool registration

# Source Code Handler Tool

## Overview
The Source Code Handler tool is designed to facilitate managing and processing source code in a structured manner.

## Features
- **Local File Management**: Allows users to list, read and write source files efficiently.

## Requirements
- Latest QT-6 framework

## Installation
1. Clone the repository: `git clone https://github.com/britus/eofmcp.git`
2. Follow the setup instructions in the repository to configure the tool for your environment.
